# Weavers-Paradise

Minecraft-мод о создании, покраске и ношении выразительной одежды. Создавайте ткани, шейте наряды, окрашивайте их продвинутыми эффектами и настраивайте внешний вид гораздо глубже, чем позволяет ванильная броня.

## Возможности

<details>
<summary><strong>Прялка Дженни</strong></summary>

Крафтовая станция, которая перерабатывает сырье в катушки соответствующего материала.

</details>

<details>
<summary><strong>Станция ткачества</strong></summary>

Крафтовая станция, которая перерабатывает катушки в ткань. Внутри есть небольшая мини-игра: чем лучше результат, тем выше качество ткани.

</details>

<details>
<summary><strong>Стол изготовления красителей</strong></summary>

**Стол изготовления красителей** используется для создания **Бутилированного красителя**, который позволяет окрашивать одежду более продвинутыми и необычными способами.

**Ядро красителя** определяет поведение красителя. Некоторые типы красителей являются простыми цветами, а другие анимируются, реагируют на мир, используют шейдеры или меняются в зависимости от игрока.

### Типы красителей

| Тип красителя             | Описание                                                                                      |
|---------------------------|-----------------------------------------------------------------------------------------------|
| **Стандартный**           | Базовый краситель, отображающий один статический цвет.                                        |
| **Прайд**                 | Использует маску прайд-флага поверх стандартной текстуры.                                     |
| **Редстоун**              | Использует красный цвет, определяемый уровнем силы от 0 до 15.                                |
| **Светокамень**           | Использует выбранный цвет с полной яркостью, из-за чего он выглядит ярким в темноте.          |
| **Лампа**                 | Объединяет поведение редстоуна и светокамня, а яркость зависит от силы редстоуна.              |
| **Скалк**                 | Пульсирует, когда рядом с игроком срабатывают звуки.                                          |
| **Цветной скалк**         | Работает как скалк, но использует выбранные цвета.                                            |
| **Голод**                 | Смешивает два цвета в зависимости от голода игрока.                                           |
| **Здоровье**              | Смешивает два цвета в зависимости от здоровья существа.                                       |
| **Время дня**             | Меняет цвет в зависимости от времени суток.                                                   |
| **Цветное время дня**     | Меняет цвет в зависимости от времени суток, используя выбранные цвета.                        |
| **Радуга**                | Сдвигает цвета со временем, создавая RGB-эффект.                                              |
| **Биом**                  | Использует цвета листвы из текущего биома.                                                    |
| **Эндер**                 | Отображается схоже с порталом в Энд.                                                          |
| **Скорость**              | Смешивает два цвета в зависимости от скорости движения.                                       |
| **Высота (бедрок)**       | Смешивает два цвета в зависимости от расстояния до бедрока.                                   |
| **Высота (уровень моря)** | Смешивает два цвета в зависимости от расстояния до уровня моря.                               |
| **Невидимый**             | Ничего не отображает.                                                                         |
| **Статика**               | Отображается как статический шум.                                                             |
| **Кристалл**              | Смещает пиксели с помощью шума Вороного.                                                      |
| **Негатив**               | Применяет эффект негатива после освещения.                                                    |
| **Истинный негатив**      | Применяет эффект негатива до освещения.                                                       |
| **Туманность**            | Отображает фиолетовую туманность.                                                             |
| **Полихроматический**     | Сдвигает оттенок в зависимости от угла и позиции камеры.                                      |

</details>

<details>
<summary><strong>Покрасочная бочка</strong></summary>

Станция для окрашивания предметов с помощью бутилированных красителей и трафаретов. Некоторые трафареты работают только с определенными предметами, что указано в подсказке трафарета.

</details>
