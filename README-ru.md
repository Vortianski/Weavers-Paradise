## Функции

<details><summary>Spinning Jenny</summary>
Этот станок позволяет перерабатывать определённые предметы в катушки соответствующего материала.
</details>

<details><summary>Clothcrafting Station</summary>
Станок для обработки катушек любого материала в ткань. Включает небольшую мини-игру, делающую процесс крафта более интересным.
</details>

<details>
<summary><strong>Dyemaking Table</strong></summary>

**Стол красильщика** - это станок для создания **Бутылочного красителя**, который позволяет окрашивать одежду продвинутыми и уникальными способами.

**Ядро красителя (Dye Core)** определяет поведение красителя. Разные типы красителей могут значительно изменять, как цвет отображается или анимируется на окрашенных предметах.

### Типы красителей

| Тип красителя      | Описание                                                                                                                         |
|-------------------|----------------------------------------------------------------------------------------------------------------------------------|
| **Default**        | Базовый краситель, отображает один статический цвет.                                                                             |
| **Rainbow**        | Переключается между всеми цветами радуги, меняется каждый тик.                                                                   |
| **Polychromatic**  | Отображает базовый цвет и применяет шейдер, который смещает оттенок в зависимости от угла обзора.                                |
| **Ender**          | Визуально имитирует эффект портала в Энд.                                                                                        |
| **Sculk**          | Использует основной цвет как базовый, а вторичный - для пульса, который срабатывает при звуках рядом с игроком.                  |
| **Biome**          | Автоматически использует цвет травы текущего биома.                                                                              |
| **Glowstone**      | Отображает базовый цвет с полной яркостью, создавая эффект свечения в темноте при использовании модов с динамическим освещением. |
| **Redstone**       | Отображает красный цвет. Можно добавить редстоун до 15 раз, чтобы усилить интенсивность.                                         |
| **Lamp**           | Комбинирует поведение Redstone и Glowstone, излучает свет в зависимости от силы сигнала ядра красителя.                          |
| **Speed**          | Цвет меняется в зависимости от скорости движения игрока, используя первичный и вторичный цвета.                                  |
| **Height (Bedrock)** | Плавный переход цвета от первичного на минимальной высоте мира до вторичного на максимальной высоте.                             |
| **Height (Sea Level)** | Переход цвета рассчитывается относительно уровня моря.                                                             |
| **Pride (16)**     | Отображает узор флага Pride. Эти красители не используют и не хранят стандартные цвета.                                          |

</details>

<details><summary>Dyeing Barrel</summary>
Станок для окрашивания предметов с использованием трафаретов и красителей. Некоторые трафареты можно применять только к определённым предметам, что указано в подсказке трафарета.
</details>
