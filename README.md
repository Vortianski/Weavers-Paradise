# Weavers-Paradise

A Minecraft mod about crafting, dyeing, and wearing expressive clothing. Create fabrics, sew outfits, dye them with advanced effects, and customize your look far beyond vanilla armor.

## Features

<details>
<summary><strong>Spinning Jenny</strong></summary>

A crafting station used to process raw materials into spools of the corresponding material.

</details>

<details>
<summary><strong>Clothcrafting Station</strong></summary>

A crafting station used to turn spools into fabric. It includes a small minigame, where better performance gives you higher-quality fabric.

</details>

<details>
<summary><strong>Dyemaking Table</strong></summary>

The **Dyemaking Table** is used to create **Bottled Dye**, which lets you dye clothing in more advanced and unusual ways.

A **Dye Core** defines how the dye behaves. Some dye types are simple colors, while others animate, react to the world, use shaders, or change based on the player.

### Dye Types

| Dye type               | Description                                                                                                       |
|------------------------|-------------------------------------------------------------------------------------------------------------------|
| **Default**            | A basic dye that displays one static color.                                                                       |
| **Pride**              | Uses a pride flag mask over the default texture.                                                                  |
| **Redstone**           | Uses a red color determined by power level, from 0 to 15.                                                         |
| **Glowstone**          | Uses the selected color at full brightness, making it appear bright in the dark.                                  |
| **Lamp**               | Combines Redstone and Glowstone behavior, with brightness controlled by redstone power.                           |
| **Sculk**              | Pulses when sounds are triggered near the player.                                                                 |
| **Colored Sculk**      | Works like Sculk, but uses selected colors.                                                                       |
| **Hunger**             | Blends between two colors based on the player's hunger.                                                           |
| **Health**             | Blends between two colors based on an entity's health.                                                            |
| **Day Time**           | Changes color based on the time of day.                                                                           |
| **Colored Day Time**   | Changes color based on the time of day while using selected colors.                                               |
| **Rainbow**            | Shifts colors over time to create an RGB effect.                                                                  |
| **Biome**              | Uses foliage colors from the current biome.                                                                       |
| **Ender**              | Renders similarly to an End Portal.                                                                               |
| **Speed**              | Blends between two colors based on movement speed.                                                                |
| **Height (Bedrock)**   | Blends between two colors based on distance from bedrock.                                                         |
| **Height (Sea Level)** | Blends between two colors based on distance from sea level.                                                       |
| **Invisible**          | Renders nothing.                                                                                                  |
| **Static**             | Renders as static noise.                                                                                          |
| **Crystal**            | Shifts pixels using Voronoi noise.                                                                                |
| **Negative**           | Applies a negative effect after lighting.                                                                         |
| **True Negative**      | Applies a negative effect before lighting.                                                                        |
| **Nebula**             | Renders a purple nebula effect.                                                                                   |
| **Polychromatic**      | Shifts hue based on camera angle and position.                                                                    |

</details>

<details>
<summary><strong>Dyeing Barrel</strong></summary>

A station used to dye items with bottled dyes and stencils. Some stencils only work on specific items, which is shown in the stencil tooltip.

</details>
